# Required Assets

Please add the following assets to the `/public` directory:

1. **logo.png** - Main logo image for the header (recommended size: 48x48px or larger)
2. **home-bg.jpg** - Full-screen background image for the home page (recommended size: 1920x1080px or larger)

These assets are referenced in:
- Header: `/logo.png`
- Home page: `/home-bg.jpg`

Until these assets are provided, placeholder images may be displayed.
